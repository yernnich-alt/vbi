# 🛡️ VBI Terminal PRO MAX

**Real-Time Reputation & Risk Monitoring System**  
Автоматизированная OSINT-платформа для мониторинга репутации компаний и брендов.

---

## 🔹 Функции
- Сбор новостей и сигналов через SerpAPI  
- Sentiment-анализ: Positive / Neutral / Negative  
- Классификация рисков: Legal, Financial, PR, Technical, Market  
- Графики: sentiment pie, risk bar, timeline  
- История сигналов в SQLite + тренды  
- Автоматические Alerts при высоком негативе
